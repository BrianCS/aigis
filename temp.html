Aigis is a style guide generator.

Make easier and maintainable.

How about it?

Aigis is a Node.js package that parses comments in your CSS and generate style guide.It is similar to Hologram. (like a Node.js implementation of Hologram.)

How does it work?

Here is an example documentation block.Markdown code blocks that will be rendered actual HTML output and in order to show how the example was coded.

/*
---
name: Buttons
category:
  - mod/btn
  - base
tag:
  - latest
  - base
---

Button styles.

* Base button style.
* Use `a` or `button` tag.

```html
<a class="ag-btn">Button</a>
<button class="ag-btn ag-btn--primary">Button</button>
```
*/
